Facebook UID API Documentation

This API converts a Facebook account username to its unique user ID (UID) using the Facebook Graph API.

API Endpoint

GET /api

Example: https://your-vercel-project.vercel.app/api?username=zuck&token=FACEBOOK_ACCESS_TOKEN&key=YOUR_API_KEY

Request Parameters

Parameter Type Required Description
username string Yes The Facebook username to lookup (letters, numbers, dots, underscores only)
token string Yes Valid Facebook Graph API access token
key string Yes API secret key for authorization (must match server's API_KEY)

Authentication

Provide your secret API key as either a query parameter key or in the HTTP header x-api-key.

Response Format

On success:

{
  "status": "success",
  "uid": "1234567890"
}

On error, the response contains a status and message describing the issue:

{
  "status": "error",
  "message": "Description of the error"
}

HTTP Status Codes

Status CodeDescription
200Success, UID returned
400Bad request (missing or invalid parameters)
403Unauthorized (invalid API key)
404UID not found
405Method Not Allowed (only GET supported)
429Too Many Requests (rate limit exceeded)
500Server error

Rate Limiting

Each IP address is limited to 10 requests per minute to protect the API from abuse.
