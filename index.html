📘 Facebook UID API

This API extracts the numeric Facebook UID from a Facebook profile URL — no access token required.

📡 Endpoint

GET /api/uid?url=https://facebook.com/zuck

🔄 Query Parameters

  • url (required): Full Facebook profile URL (e.g. https://facebook.com/zuck)

✅ Success Response

{
  "status": "success",
  "uid": "4"
}

❌ Error Response

{
  "status": "error",
  "error": "Invalid Facebook profile URL"
}

⚡ Example Usage

curl "https://your-vercel-url.vercel.app/api/uid?url=https://facebook.com/zuck"

🔐 Rate Limits

Currently unlimited. Add protection if deploying to public.
